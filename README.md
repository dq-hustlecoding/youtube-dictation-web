## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## How to Use

1. Paste a YouTube URL
2. Click "Start Practicing"
3. Watch the video segment
4. Type what you hear
5. Check your answer and get accuracy score
6. Move to the next subtitle
